# E-Commerce Back End

## Description
This app will allow you to use mySql for a backend E-Commerce site. 

## Installation 
- Clone repository
- npm init
- npm install mysql2
- npm install sequalize

## Usage
- mysql -u root -p 
- run schema
- npm run seed
- npm start
